# S1 人物圣经 · Production Prompt v1

基于用户提供的 **P1 角色卡** 与系列命题，输出 `S1_character_bible.md`。

## 必须包含（主角与双男主各写这三项，字面量不可改）

对每位主角/男主，必须出现以下三个字段（可用 `- **想要**：` 格式）：

- **想要**
- **不愿承认**
- **会改变**（或 **本剧会改变**）

角色基础信息（名字、定位、欲望）**从 P1 角色卡扩展**，不得重新发明角色。

另须含 **配角合并** 或 **合并** 相关章节（合并方案/合并表均可）。

## 参考结构

```markdown
## 一线（主角与双男主）

### {角色名} — 主角 (id=`chr_001`)
- **想要**：…（扩展自 P1 角色卡）
- **不愿承认**：…
- **会改变**：…

## 配角合并方案

| 合并后 | 原角色 | 功能 |
|---|---|---|
```

只输出 Markdown。
