# S3 分集清单 · Production Prompt v1

为单季输出 `episode_list.md`，把 `S2_season_map.md` 中该季一行切成可拍短集。每集自成小闭环：**核心冲突 + 主角的一个选择 + 集尾钩子**。

## 语言

- **全文中文**（标题、表头、集情、冲突、选择、钩子）。
- 若需引用原著**对白或独白**，保留**英文原文**，不得改写成中文释义。
- 人名、地名用与 S1 人物圣经一致的译名；西幻专有名词可保留常见译音。

## 必须遵守的表格格式（列顺序不可调换）

```markdown
# 第{N}季分集清单 · {季ID}「{季命题关键词}」（Ch{起}–{止}）

> S3 产出（{季ID}）。竖屏短剧，单集 2–5 分钟。

## 季命题回顾
{从 S2 提炼本季主角蜕变与引擎，1–2 句}

## 分集总表

| 集 | 一句话集情 | 覆盖 | 核心冲突 | 主角的选择 | 集尾钩子 |
|---|---|---|---|---|---|
| **EP01** | … | Ch1 | … | … | … |
| **EP02** | … | Ch2 | … | … | … |
```

## 硬性规则

- 集号必须是 `**EP01**`、`**EP02**`…（禁止 `Episode 1`）
- 「覆盖」列用 `Ch{n}` 或 `Ch{n}–{m}` 格式，须落在本季章节范围内
- 表头必须是中文六列：**集 | 一句话集情 | 覆盖 | 核心冲突 | 主角的选择 | 集尾钩子**（禁止英文 Logline / Conflict / Choice / Cliffhanger）
- 每集「一句话集情」≤40 字，锋利可拍
- 「主角的选择」须写清**主动抉择**（拒、冲、赌、揭、忍等），禁止被动旁观
- 「集尾钩子」须**视觉化**（特写、新威胁入画、身体异变等），禁止叙述性总结
- 章节覆盖须连续、无大缺口；可 1 集覆盖 1–2 章，按冲突密度切分
- 角色名、CP、世界观必须与 S1/S2 输入一致，不得替换为其他作品人设

只输出 Markdown，不要解释。
