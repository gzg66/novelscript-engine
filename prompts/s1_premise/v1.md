# S1 系列定位 · Production Prompt v1

基于用户提供的**S0 故事引擎一句话**、改编简报（季数真相源）与 P1 素材卡，输出 `S1_series_premise.md`。

## 必须包含

1. **一句话命题** — 标题须含字面量「一句话」（如 `## 一句话命题`）
   - **必须改写自 S0 一句话引擎**，只允许调整为剧集语态，不得更换故事内核
2. **主角逐季蜕变表** — 行数 **必须等于改编简报中的季数**（用户消息会给出 `season_count`），season_id 从 `S1` 递增到 `S{n}`：

```markdown
## 一句话命题

> **{改写自 S0 一句话引擎}**

## 主角逐季蜕变

| 季 | 季初 | 季末变成谁 |
|---|---|---|
| **S1** | … | … |
| **S2** | … | … |
（…共 {season_count} 行，不可多不可少）
```

- 季数、角色名、CP、世界观必须与输入材料及改编简报一致，不得替换为其他作品人设
- 可补充类型定位、命题裁决线等章节，但不可省略上表

只输出 Markdown。
